Keyboard Navigation and Shortcuts

To improve accessibility, we have implemented keyboard navigation and shortcuts. Use the following keys to navigate through the application:

  • Tab: Move to the next interactive element
  • Shift + Tab: Move to the previous interactive element
  • Enter: Activate the selected element
  • Esc: Close the current dialog or menu
